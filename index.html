How to Install

  1. Click the Download button above to get the extension.zip file.
  2. Unzip the file to a permanent folder on your computer.
  3. Open Google Chrome and navigate to chrome://extensions.
  4. Enable Developer Mode using the toggle in the top-right corner.
  5. Click Load unpacked and select the folder where you unzipped the extension.
  6. You're ready! Right-click any link to start previewing.
